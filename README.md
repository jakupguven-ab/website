# Jakup Güven AB — business website

Public website for Jakup Güven AB's AI automation business: narrow vertical AI agents
for Swedish småföretagare, human-in-the-loop by design.

**Live:** https://jakupguven-ab.github.io/website/

Fully static (plain HTML + CSS, no build step). Served by GitHub Pages from `main` /
root. Swedish-first. Contact: info@jakupguven.se.

This repo is public — nothing non-public goes in it, including comments and commit
messages. Team conventions and content scope: see `CLAUDE.md`.
